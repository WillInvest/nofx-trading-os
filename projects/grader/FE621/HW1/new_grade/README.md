# FE621 HW1 — New Grade (detailed)

This folder holds the **same detailed per-student grading memos** as `../grade/`. Each file is a full grading report for one student.

## Contents

- One Markdown file per student: `{lastname_firstname}.md`.
- **INDEX.md** — Quick reference: student name and final score only.

Each memo includes:

- **Final score** (base + bonus)
- **Part 1–4** with question-level sub-scores (e.g. Q1 4/5, Q6 5/7) and comments
- **Strengths / Weaknesses / AI assessment**

## Sync

Content is synced from `../grade/`. For the single source of truth, use `../grade/` or this folder interchangeably.
