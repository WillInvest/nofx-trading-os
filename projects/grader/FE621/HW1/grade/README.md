# FE621 HW1 — Grade (detailed)

This folder contains **per-student grading memos** for FE621 Homework 1. Each file is a detailed grading report for one student.

## Contents

- One Markdown file per student: `{lastname_firstname}.md` (e.g. `kelleher_jackson.md`).
- Each file includes:
  - **Final score** (base + bonus)
  - **Part 1 (20 pts)**: Data gathering — Q1–Q4 with sub-scores and comments
  - **Part 2 (50 pts)**: Data analysis — Q5–Q12 with sub-scores and comments
  - **Part 3 (30 pts)**: AMM fee revenue — Q3a–Q3c with sub-scores and comments
  - **Part 4**: Bonus (10 pts) if attempted
  - **Strengths / Weaknesses / AI assessment**
